## 1. Describe the human problems
// Problems (Pain Points)
    - มีพฤติกรรมการนอนไม่เป็นเวลา ทำให้นอนดึกและเข้านอนได้ยาก
    - ตื่นยาก แม้ตั้งนาฬิกาปลุกหลายอันก็ยังหลับต่อ ส่งผลให้ไปเรียนหรือทำงานสาย
    - ต้องพึ่งพาคนใกล้ตัว เช่น แฟนหรือครอบครัว ให้ช่วยโทรปลุกทุกวัน ทำให้รู้สึกเกรงใจและเป็นภาระของผู้อื่น

## 2. Describe how your solution solves these problems
// Solutions
    1. Host Matching
    ผู้ใช้เลือกโฮสต์ที่เหมาะกับบุคลิกและสไตล์การพูดคุยของตนเอง

    2. Personalized Experience
    ผู้ใช้กำหนดเวลาเข้านอน เวลาปลุก และรูปแบบการดูแลได้ตามต้องการ

    3. Subscription Service
    สมัครใช้งานแบบรายสัปดาห์หรือรายเดือน เพื่อให้ได้รับบริการอย่างต่อเนื่อง

    4. Bedtime Reminder
    โฮสต์โทรเตือนก่อนเวลาเข้านอน พร้อมพูดคุยสั้น ๆ เพื่อช่วยให้ผู้ใช้เตรียมตัวพักผ่อน

    5. Wake-up Calls & Confirmation
    โฮสต์โทรปลุก และยืนยันว่าผู้ใช้ตื่นแล้ว หากยังไม่รับสายสามารถโทรซ้ำตามจำนวนครั้งที่กำหนด

    6. Progress Tracking
    แสดงสถิติการเข้านอน การตื่นตรงเวลา และความต่อเนื่องของการสร้างวินัย
